UBC Sanskrit Digital Philology ruby tools. This gem will eventually be broken apart into individual component gems.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'dphil'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install dphil
